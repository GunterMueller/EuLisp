4 Installation

  • Edit Makefile
  • Add the mpi library load path, e.g. -L/net/dominica/denton_export/MPI/mpich/lib/IRIX/ch_p4 to CFLAGS in your ~/\.eulrc.${ARCH}
  • Bin.${ARCH}/youtoo.sh -ar mpis -l level1
  • make
  • The headers in the examples will give you enough information how to compile them.
